# Vue.js To-Do List Application!

A simple to-do list app created with only Vue.js!

## How to Run this App
```
npm install
npm run serve
```

### Compiles and Minifies for Production
```
npm run build
```

### Lints and Fixes Files
```
npm run lint
```
